## Notas de Generación

### Densidad Óptima
- **Bypass fácil:** 50-80 palabras
- **Balance:** 100-150 palabras
- **Máximo detalle:** 150-200 palabras
- **Riesgo de rechazo:** 200+ palabras con tokens sensibles

### Orden de Importancia
1. Entidad core (siempre aparece)
2. Criaturas cerebrales (alto impacto visual)
3. Iluminación (define mood)
4. Texturas (define calidad)
5. Detalles de fondo (pueden perderse)

### Tokens de Alta Efectividad
Estos tokens consistentemente generan el resultado esperado:
- `BULGING OVERSIZED EYES`
- `fleshy tendrils`
- `pulsating`
- `organic decay`
- `toxic green glow`
- `arterial crimson`

### Tokens a Evitar
Estos tokens frecuentemente causan rechazo o resultados no deseados:
- `blood` → usar `vital fluid`
- `gore` → usar `visceral texture`
- `violent` → usar `intense`, `primal`
- `drugs` → usar `herbal sacrament`
- `death` → usar `entropy`, `decay`
