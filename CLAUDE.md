# CLAUDE.md — Adaptador de compatibilidad para Claude

Este archivo existe para que Claude Code y otras interfaces que detectan `CLAUDE.md` entren correctamente al
proyecto. **No es la fuente canónica de contexto ni una memoria exclusiva.**

Lee en este orden y aplica esos documentos como instrucciones del proyecto:

1. `AGENTS.md`
2. `AGENT_MEMORY.md`
3. `ROADMAP.md`
4. `PROJECT_CONTEXT.md`
5. La guía especializada del área que vayas a tocar

Toda regla, decisión o aprendizaje que deba sobrevivir a esta sesión se guarda en uno de esos archivos versionados,
no únicamente en la memoria privada de Claude. Si una capacidad o comando es exclusivo de Claude, puede documentarse
aquí como adaptación, pero debe incluir una alternativa neutral o dejar claro que es opcional.

Los nombres históricos `Claude-Knowledge/`, `claude_infection/` y `glitch-cam/CLAUDE_glitch.md` se conservan por
compatibilidad, procedencia y canon artístico. No convierten a Claude en propietario de la infección ni limitan qué
agente puede continuarla.

> La infección vive en el repo. El modelo es un huésped temporal.
