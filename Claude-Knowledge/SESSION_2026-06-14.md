## 16. Desktop State Persistence — la PC nunca se apaga

**Commits:** `3780c39`, `a4f3de2`, `96c29b5`, `2b47001`

| Componente | Detalle |
|------------|---------|
| **`saveDesktopState()`** | Captura todas las `.win` del DOM (id, title, icon, appFile, x, y, w, h, min). Throttle 400ms o force sync en beforeunload |
| **`restoreDesktopState()`** | Al boot (800ms delay), lee `esquizoDesktopState` de localStorage y reabre cada ventana con `openApp(file, title, icon, {w, h, x, y})` |
| **`clearDesktopState()`** | Limpia el estado al hacer shutdown |
| **Hooks** | Guarda en: open, close, minimize, drag end, resize end, beforeunload |
| **Datos** | Se almacenan como `data-*` attributes en cada `.win` (accesibles desde fuera del closure WM) |

### Resize 8-zone (como Windows):

| Zona | Cursor | Dirección CSS |
|------|--------|-------------|
| Borde superior | `n-resize` | `.rz-n` |
| Borde inferior | `s-resize` | `.rz-s` |
| Borde izquierdo | `w-resize` | `.rz-w` |
| Borde derecho | `e-resize` | `.rz-e` |
| Esquina NO | `nwse-resize` | `.rz-nw` |
| Esquina NE | `nesw-resize` | `.rz-ne` |
| Esquina SO | `nesw-resize` | `.rz-sw` |
| Esquina SE | `nwse-resize` | `.rz-se` |

Mínimo 280x160. Desactivado en diálogos, ventanas maximizadas y mobile. Cada zona tiene su lógica de delta (n/s/e/w/ne/nw/se/sw) que ajusta width, height, left, top.
